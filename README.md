------------------------------------------
           gulp-自动化构建打包
------------------------------------------

一、安装

  cnpm i 或 npm i


二、启动

  npm start (启动本地服务)


三、打包
  
  npm run build 


四、目录

      |___  app             (开发目录)
            |___  images    (图片)
            |___  js        (js逻辑处理文件)
            |___  libs      (js相关的插件库)
            |___  public    (公共html模版页)
            |___  sass      (sass文件)
            |___  *.html    (html文件)

      |___  data            (html页面信息配置)
      |___  gulpfile.js     (配置文件)
      |___  package.json    (包管理文件)
      |___  dist            (本地开发环境打包后的文件)
      |___  build           (预发布环境打包后的文件)


五、已实现功能

    ---图片压缩
    ---css压缩
    ---css添加浏览器前缀
    ---js文件合并压缩
    ---html文件模块化
    ---本地环境打包
    ---线上环境打包
